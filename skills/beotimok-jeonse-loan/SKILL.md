# 버팀목 전세자금대출 (주택도시기금)

무주택 서민 전세자금 대출. 일반·청년·신혼 3종.

## Workflow

### 1. 적합한 유형 판별

```
1. 연령 확인
   → 만 19~34세: 청년전용 우선 (금리 유리)
   → 35세+: 일반 또는 신혼

2. 혼인 여부
   → 혼인 7년 이내: 신혼부부 (한도·금리 유리)
   → 미혼 34세 이하: 청년전용
   → 미혼 35세+: 일반

3. 소득 확인
   → 5천만 이하: 일반·청년 모두 → 금리 비교
   → 5천~7.5천만: 신혼만 가능
   → 7.5천만+: 기금대출 불가 → 시중은행

4. 출산 2년 이내?
   → YES: 신생아 특례 버팀목 검토 (더 유리)
```

### 2. 3종 비교표

| 항목 | 일반 | 청년전용 | 신혼부부 |
|------|------|---------|---------|
| **연령** | 무관 | 19~34세 | 혼인7년 |
| **소득** | 5천만 | 5천만 | 7,500만 |
| **한도(수도권)** | 1.2억 | 1.5억 | 2.2억 |
| **보증금 비율** | 70% | 80% | 80% |
| **금리** | 2.5~3.5% | 2.2~3.3% | 1.9~3.3% |

### 3. 금리 안내 (2026.02 기준)

| 소득 | 일반 | 청년 | 신혼 |
|------|:----:|:----:|:----:|
| ~2천만 | 2.5% | 2.2% | 1.9% |
| 2천~4천만 | 2.8% | 2.5% | 2.2% |
| 4천~5천만 | 3.1% | 2.8% | 2.5% |
| 5천~6천만 | 3.5% | 3.3% | 2.8% |

> 지방 소재: **0.2%p 인하**

### 4. 주의사항

- 기간: 2년 (최장 10년 연장)
- 전입 1개월 내, 실거주 필수
- 기금대출 중복 불가 (디딤돌과 동시 X)
- HUG/HF 보증서 필요
