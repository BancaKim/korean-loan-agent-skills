# DSR (총부채원리금상환비율) 계산기

## Workflow

### 1. 사용자 입력 수집

다음 정보를 사용자에게 확인:
- **필수**: 연소득(만원), 신규 대출금액(만원), 금리(%), 상환기간(년/개월)
- **선택**: 상환방식, 금리유형(변동/혼합5년/혼합10년/고정), 지역(수도권/지방), 금융권(은행/2금융), 기존대출

### 2. 계산 실행

`calculator.py`를 CLI로 실행합니다.

**DSR 계산:**
```bash
python [SKILLS_DIR]/dsr-calculator/references/calculator.py calculate \
  --annual-income 5000 \
  --loan-amount 30000 \
  --loan-rate 4.5 \
  --loan-months 360 \
  --rate-type 변동 \
  --region 수도권
```

**최대 대출 가능액 (상환방식별/기간별/금리유형별 비교 포함):**
```bash
python [SKILLS_DIR]/dsr-calculator/references/calculator.py max-loan \
  --annual-income 5000 \
  --loan-rate 4.5 \
  --loan-months 360 \
  --rate-type 변동 \
  --region 수도권
```

출력은 JSON이며, `by_method`, `by_period`, `by_rate_type` 필드로 비교표를 제공합니다.

### 3. 결과 해석

- `stress_rate`: 스트레스 가산 적용 후 실제 DSR 산정 금리
- 변동금리 vs 고정금리 차이가 크면 금리유형 전환 추천
- DSR 초과 시: 고정금리 전환(가장 효과적), 만기 연장, 기존대출 상환 안내

## 핵심 규제 데이터 (계산기에 내장)

### DSR 한도
| 금융권 | DSR 한도 | 적용 기준 |
|--------|:-------:|----------|
| 은행권 | **40%** | 총 대출잔액 1억 초과 |
| 2금융권 | **50%** | 총 대출잔액 1억 초과 |

### 스트레스 가산금리 (3단계, 2025.7.1~)
| 구분 | 기본 가산금리 |
|------|:----------:|
| 수도권/규제지역 주담대 | **+3.0%p** |
| 지방 주담대 | **+0.75%p** |
| 신용대출 (1억 초과) | **+1.5%p** |

### 금리유형별 스트레스 적용비율
| 금리유형 | 적용비율 | 수도권 실제 가산 |
|---------|:------:|:-----------:|
| 변동금리 (5년 미만) | 100% | **+3.0%p** |
| 혼합형 (고정 5~9년) | 80% | +2.4%p |
| 혼합형 (고정 9~15년) | 60% | +1.8%p |
| 순수 고정금리 | 0% | **+0.0%p** |

### 만기일시상환 의제 산정만기
| 대출 유형 | 산정만기 상한 |
|----------|:----------:|
| 주택담보대출 | 10년 |
| 신용대출 | 5년 |
| 카드론 | 3년 |
