# 신생아 특례 디딤돌대출 (주택구입)

2년 내 출산(입양) 가구 초저금리 주택구입 대출. 디딤돌 기반이나 조건 대폭 완화.

## Workflow

### 1. 자격 판별

```
1. 2년 내 출산/입양? → NO: 일반 디딤돌 안내
2. 소득 1.3억 이하? (맞벌이 2억) → NO: 대상 아님
3. 주택가격 9억 이하? → NO: 대상 아님
4. 소득 6천만 이하 + 주택 5억 이하? → 디딤돌과 금리 비교
```

### 2. 디딤돌 vs 신생아 특례 비교

| 항목 | 디딤돌 | 신생아 특례 |
|------|--------|-----------|
| **소득** | 6천만 | **1.3억** (맞벌이 2억) |
| **주택가격** | 5억 | **9억** |
| **최대한도** | 3.2억 | **4억** |
| **최저금리** | 2.55% | **1.80%** |
| **추가출산** | 없음 | **-0.2%p + 5년 연장** |

### 3. 금리 안내 (특례금리)

| 소득 | 10년 | 15년 | 20년 | 30년 |
|------|:----:|:----:|:----:|:----:|
| ~2천만 | **1.80%** | 1.90% | 2.00% | 2.05% |
| 2천~4천만 | 2.15% | 2.25% | 2.35% | 2.40% |
| 4천~6천만 | 2.40% | 2.50% | 2.60% | 2.65% |
| 6천~8.5천만 | 2.65% | 2.75% | 2.85% | 2.90% |
| 8.5천~1.3억 | 2.90~3.50% | | | |
| 맞벌이 1.3~2억 | 3.50~3.80% | | | |

### 4. 추가출산 우대 안내

```
기본: 5년 특례금리
추가 출산 시: -0.2%p + 5년 연장 (자녀당)
최장: 15년 특례
→ 특례 종료 후 디딤돌 일반금리 전환
```

### 5. 대환대출

기존 주담대 → 신생아특례 전환 가능. **1주택 세대주도 가능**.

## 관련 스킬
- `didimdol-loan`: 일반 디딤돌 (출산요건 없이 저금리)
- `newborn-jeonse-loan`: 전세자금 버전
- `bogeumjari-loan`: 고정금리 장기
- `loan-affordability`: LTV·DTI·DSR 동시 적용
